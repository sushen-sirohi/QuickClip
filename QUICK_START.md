# 📋 Clipboard Manager - Quick Start

## 🚀 First Launch

When you start the app:
- A tray icon appears in your menu bar/system tray
- The app runs silently in the background
- Your clipboard is being monitored automatically

## ⌨️ Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+V` (Mac)<br>`Ctrl+Shift+V` (Win/Linux) | Toggle window visibility |
| `ESC` | Hide window |
| `Cmd+W` / `Ctrl+W` | Hide window |
| Type in search box | Filter clipboard history |

## 🎯 How to Use

### Show the Clipboard Manager
Press `Cmd+Shift+V` (or `Ctrl+Shift+V` on Windows/Linux)

### Search Your History
Just start typing in the search box - results filter instantly!

### Copy an Item
**Method 1:** Click any item - it copies and window auto-hides
**Method 2:** Hover and click the "Copy" button - window stays open

### Delete an Item
Hover over an item and click the "Delete" button

### Hide the Window
- Press `ESC` (fastest)
- Press `Cmd+W` or `Ctrl+W`
- Press `Cmd+Shift+V` / `Ctrl+Shift+V` again
- Click an item (copies and hides)

### Clear All History
Click the red "Clear All" button (you'll get a confirmation prompt)

## 💡 Tips

**Window Won't Auto-Hide**
- This is by design! You have full control
- Window stays visible until YOU close it
- No more accidental closes when clicking buttons

**Fast Workflow**
1. `Cmd+Shift+V` - Open
2. Type to search (optional)
3. Click item - Auto-copies and closes
4. Paste with `Cmd+V` / `Ctrl+V`

**Keep Window Open**
- Just leave it visible while you work
- Copy multiple items by clicking "Copy" button
- Close it when done with `ESC`

**Background Operation**
- App runs 24/7 in the background
- Uses minimal resources (~30-50MB)
- Always ready when you need it

## 🔧 System Tray

**Left-click:** Toggle window
**Right-click:** Open menu
- Show Clipboard Manager
- Quit

## ❓ Troubleshooting

**Window opens and closes immediately?**
- Don't hold the shortcut key
- Press and release quickly
- Wait 300ms between toggles

**Shortcut not working?**
- Another app might be using it
- Use the system tray icon instead
- Check System Preferences > Security (macOS)

**Clipboard not being recorded?**
- Restart the app
- Check if the app has clipboard permissions

## 🎓 Pro Tips

1. **Rapid Access:** Keep window open while working on repetitive tasks
2. **Search Power:** Search works on all text - very fast!
3. **Clean History:** Regularly clear old items you don't need
4. **System Tray:** Quickest way to show/hide if you forget the shortcut

## 📊 What Gets Saved?

✅ **Saved:**
- All copied text
- Timestamps
- Everything local on your machine

❌ **NOT Saved:**
- Images (text only currently)
- Files
- Passwords from password managers (usually)
- Nothing sent to any server

## 🔒 Privacy

- 100% local storage
